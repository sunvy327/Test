Simple face detection streaming app using flask and openCV

Step 1: 

Install openCV (pip3 install opencv)

Step 2: 

Install Flask (pip3 install flasK)

Step 3: 

Install Flask-pymongo (pip3 install Flask-PyMongo)

Step 4: 

Install Mongo Compass for GUI of the Database

Step 5: 

Create a database named "saveimage" so that it matches the database name with the files. 

Step 6: 

run the file "app.py"
